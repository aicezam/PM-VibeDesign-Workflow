# Ant Design Mobile 视觉与组件规范

本规范作为 `axure-prototype-svg` 技能的**单独风格模板**，用于在生成移动端 (H5/App) 页面原型时，替换默认的视觉样式规范 和组件样式，确保最终生成的 SVG 符合 Ant Design Mobile 的标准。

## 1. 视觉样式规范 (Visual Tokens)

### 1.1 字体 (Typography)
- **字体家族**: `font-family="Arial, sans-serif"`
- **字号**:
  - 大标题 (Title): 20px / 18px (加粗)
  - 列表/正文主字 (Body): 16px 或 15px
  - 辅助/次要说明 (Caption): 13px 或 12px
- **行高**: 默认 `1.5`

### 1.2 色彩 (Colors)
- **品牌主色 (Primary)**: `#1677FF`
- **中性色 (Neutral)**:
  - 页面背景 (Background): `#F5F5F9`
  - 容器/区块背景 (Surface): `#FFFFFF`
  - 主文本色 (Text Main): `#333333`
  - 次文本色 (Text Secondary): `#999999`
  - 辅助/提示文字 (Text Placeholder): `#CCCCCC`
  - 分割线/边框 (Border): `#EEEEEE` 或 `#E5E5E5`
- **功能色 (Semantic)**:
  - 成功 (Success): `#00B578`
  - 警告 (Warning): `#FF8F1F`
  - 危险/错误 (Danger): `#FF3141`

### 1.3 尺寸与间距 (Sizes & Spacing)
- **屏幕基准**: `375 x 812` 或 `390 x 844`
- **点击区域 (Touch Target)**: 最小建议 44px 高度，保证手指触控体验。
- **圆角 (Border Radius)**:
  - 卡片圆角: 8px
  - 按钮圆角: 4px (标准) / 100px (胶囊形)
- **间距 (Spacing)**:
  - 页面左右安全边距: 12px 或 16px
  - 区块间垂直间距: 12px
  - 列表项内边距: 16px (水平) x 12px (垂直)

## 2. 组件风格与使用场景 (Components)

### 2.1 导航与布局 (Navigation & Layout)
- **导航栏 (NavBar)**: 高度 45px，居中标题（17px 加粗），左侧通常为返回 `<` 图标，右侧为额外操作。背景白底或透明。
- **底部标签栏 (TabBar)**: 高度 50px，固定在底部，包含图标与文字（10px），选中项为主色，未选中为灰色。
- **标签页 (Tabs)**: 紧凑排列，选中项文字加粗且底部有主色短下划线。用于同层级多视图切换。

### 2.2 数据录入 (Data Entry)
- **按钮 (Button)**:
  - 高度: 大按钮 48px，中按钮 40px，小按钮 32px。
  - **块级主按钮 (Block Primary)**: 宽度 100%，蓝底白字，通常吸底或位于表单最下方。
  - **次按钮 (Default)**: 白底黑字，浅色边框。
- **列表项输入 (List Input)**: 移动端表单常融合在列表中。左侧标签 (Label) 占一定比例，右侧为输入框，无边框，带底部细分割线。
- **选择器 (Picker)**: 点击列表项后，从底部弹出的半屏滚轮选择器。包含顶部操作栏（取消、标题、确定）。
- **开关 (Switch)**: 宽度 40px，高度 24px，圆角 12px。开启时为主色底色带白色滑块，关闭时为灰色底色。

### 2.3 数据展示 (Data Display)
- **列表 (List)**: 移动端最常见结构。白底区块，项目之间带 `#EEEEEE` 下边框。可包含左侧图标、主标题、副标题、右侧提示文案及向右箭头 `>`。
- **卡片 (Card)**: 带有明显内外边距，圆角 8px，偶尔带微弱阴影。用于将复杂信息（如订单详情、图文摘要）分组。
- **标签 (Tag)**: 较小，高度通常 20px，用于标识状态或属性（如“已发货”、“包邮”）。
- **步骤条 (Steps)**: 常垂直或水平展示进度（如物流追踪）。

### 2.4 反馈 (Feedback)
- **轻提示 (Toast)**: 屏幕中央出现的黑色半透明圆角矩形，白字，停留 2-3 秒消失。用于成功、加载中等不打断用户流的提示。
- **对话框 (Dialog)**: 居中弹窗，带半透明蒙层，宽度约为屏幕的 80%。圆角较大，通常底部有并排或垂直的“取消”、“确定”按钮。
- **动作面板 (ActionSheet)**: 从底部弹出的菜单列表，用于提供当前上下文的多个操作选项（如“分享”、“删除”），最底部常带“取消”按钮。
- **空状态 (Empty)**: 居中的插画+文案，用于列表无数据时。
