# Axure Markdown 渲染元件

通过简单的交互配置，可将 Axure 元件中的 Markdown 文本渲染为精美的网页文档，无需安装任何插件。

## 支持功能

- 全套标准 Markdown 语法（标题、列表、代码块、引用等）
- Mermaid 图表（流程图、时序图、甘特图等）
- 兼容动态面板切换

## 快速开始

1. **创建加载器元件：**
   - 拖入一个普通的矩形或文本元件作为"加载器"
   - 设置该元件的交互：**载入时 → 打开链接 → fx **
   - 在链接框中输入以下代码：

   ```
   javascript:{$axure.utils.loadJS('https://izam.cn/js/marked.min.js');$axure.utils.loadJS('https://izam.cn/js/markdown.js');$axure.utils.loadJS('https://cdn.bootcdn.net/ajax/libs/mermaid/11.12.0/mermaid.min.js');}
   ```

2. **创建 Markdown 内容元件：**
   - 拖入一个普通的矩形或文本元件
   - 在其中输入您的 Markdown 文本
   - 在右侧的"概要(Outline)"面板中，将该元件重命名为 **`markdown`**，或 **``*MD*

3. **预览：**
   - 按 `F5` 预览即可看到渲染效果
